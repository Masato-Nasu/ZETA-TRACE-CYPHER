# ZETA TRACE CYPHER

**ZETA TRACE CYPHER** is a visual cipher experiment derived from **ZETA 2 CYPHER**.

It encrypts text first, then stores the encrypted data as one or more PNG random-walk traces.  
The visible line is not decoration.  
**The line itself is the encrypted data carrier.**

## Concept

Text walks.  
The walk becomes a line.  
The line becomes the cipher.

A message is encrypted by the ZETA 2 CYPHER core, then transformed into PNG random-walk drawings.  
Large data is automatically split into multiple PNG pages.  
During decryption, the PNG pages are loaded together and reconstructed into the original encrypted data before password-based decryption.

```text
text
→ ZETA 2 encryption
→ one or more random-walk PNG traces
→ PNG input
→ bundle reconstruction
→ decryption
→ original text
```

The drawing is not an illustration of the ciphertext.  
It is the ciphertext.

## Features

- Text encryption
- Password-based decryption
- PNG random-walk output
- Multiple PNG output for larger data
- Multiple PNG input for decryption
- Automatic bundle reconstruction
- No fixed input byte limit in the app UI
- Warning shown for input over 50000 bytes
- Minimal, no-gimmick interface
- Works entirely in the browser
- No server-side storage

## Multi-page bundle

When the encrypted carrier data is too large for one trace, ZETA TRACE CYPHER automatically splits it into multiple PNG pages.

Each page contains internal bundle information:

- bundle ID
- page index
- total page count
- payload fragment
- checksum

The page order is determined internally, not by filename.  
All pages must be selected together for decryption.

Large input is allowed, but very large data can generate many PNG pages and may take time to save, load, and decode in the browser.

## Current limitations

Supported:

- Original PNG files generated by this app
- Multiple PNG selection for bundle decryption
- Lightly re-saved or resized images in some cases

Not supported reliably yet:

- strong JPEG compression
- screenshots
- camera scanning
- rotation
- trimming
- heavily damaged image recovery

If the PNG is edited, compressed, resized, or converted too strongly, it may no longer decode.

## Security note

ZETA TRACE CYPHER is an experimental personal cipher and visual data carrier.

The PNG trace may make the ciphertext visually obscure, but the strength of decryption still depends heavily on the password.  
Use a long, unique password.

This project has not been professionally audited.  
Do not use it as the sole protection for critical secrets.
