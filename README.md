# ZETA TRACE CYPHER v0.2.0

ZETA TRACE CYPHER is a PNG random-walk carrier prototype derived from ZETA 2 CYPHER v0.5.0. v0.2.0 raises the text limit and improves the walk-cell drawing style.

The encryption core remains based on the existing ZETA 2 CYPHER structure. The difference is the carrier:

- input text is encrypted with the ZETA 2 CYPHER logic
- encrypted bytes are converted into a continuous PNG random-walk drawing
- the PNG itself can be loaded back into the app
- the app reads the line pattern from the PNG and restores the encrypted bytes
- the password then decrypts the original text

## Current scope

This first version intentionally supports only original PNG files generated by this app.

Supported:

- text to PNG random-walk cipher
- up to 6000 bytes of input text
- PNG save
- PNG load
- password-based decode
- original PNG roundtrip

Not supported yet:

- camera reading
- screenshots
- JPEG
- rotation
- resize
- cropping
- error correction

## Design note

The visible drawing is not just decoration. Each random-walk cell carries one encrypted byte. v0.2.0 uses denser 8-step cells with zeta-derived jitter, connected in a serpentine layout so the image reads visually as a more continuous walking line while remaining decodable from PNG pixels.

## Files

- `index.html` — app UI
- `styles.css` — minimal UI styling
- `app.js` — ZETA 2 encryption core plus PNG random-walk carrier
- `_headers` — cache control for Cloudflare Pages

## Deploy

If the extracted app folder `zeta-trace-cypher-v02` is directly on your Desktop, deploy with:

```powershell
cd "$env:USERPROFILE\Desktop\zeta-trace-cypher-v02"
npx wrangler pages deploy . --project-name zeta-trace-cypher
```
